# Clean House API

This project aims to provide customer data and map routes to the customer's home.

## Running Project

Before running the project, make sure you have the following requirements:

- Node.js: >=20.10.0

1. Install Dependencies:
     ```shell
     npm i
     ```
2. Create database:
     ```shell
     docker-compose up -d
     ```

3. Running application:
     ```shell
     npm run dev
     ```
